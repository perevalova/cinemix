# Cinemix
Movie catalog

## Installation

#### 1. Install dependencies:

```bash
pip install -r requirements.txt
```

#### 2. Implement migrations:

```bash
python manage.py migrate
```

#### 3. Run sever:

```bash
python manage.py runserver
```
